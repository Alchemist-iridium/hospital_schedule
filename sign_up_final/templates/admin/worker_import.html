File Format Requirements

The CSV or XLSX file must contain the following columns with the specified data types:

Column Name Data Type Description
id Integer Unique identifier for the worker (e.g., 101, 102)
name String Worker’s full name (e.g., "John Doe")
type String Type of worker ("senior" or "junior")
initial_order Integer Initial order for scheduling (e.g., 1, 2, 3)
password String Worker’s initial password (e.g., "pass123")

Note: Ensure all values are provided; missing or invalid data will result in errors during import.
